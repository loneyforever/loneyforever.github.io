最小表示法

算法解释

我们这里要i = 0,j = 1,k = 0,表示从i开始k长度和从j开始k长度的字符串相同（i，j表示当前判断的位置）。

当我们a[i] == a[j]时，根据上面k的定义，我们的需要进行k+1操作。

当a[i] >a[j]时，我们发现i位置比j位置上字典序要大，那么不能使用i作为开头了，我们要将i向后移动，移动多少呢？有因为i开头和j开头

的有k个相同的字符，那么就执行 i = i + k +1，相反a[i] < a[j]时，执行：j = j + k +1 ，最终i和j中较小的值就是我们最终开始的位置。

代码实现：

#incude<iostream>
using namespace std;

void get_min(int a[]){
    //获取循环数组a的最小表示
    int n = a.size();
    vector<int> b(2n);
    for(int i = 0; i < 2n; i++) b[i] = a[i % n];
    int i = 0, j = 1, k;
    while(i < n && j < n)
    {
        for(k = 0; k < n && b[i + k] == b[j + k]; k++);//寻找不相同的位置 
        if(k == n) break;//如果k等于n，那么数组a就由一个相同元素构成 
        if(b[i + k] < b[j + k])
        {
            j = j + k + 1;
            if(i == j) j++;//i和j相同时有一个数要+1
        }
        if(b[i + k] > b[j + k])
        {
            i = i + k + 1;
            if(i == j) i++;//i和j相同时有一个数要+1
        }
    }
    k = min(i, j);
    for(i = 0; i < n; i++) a[i] = b[i + k];
} //get_min
