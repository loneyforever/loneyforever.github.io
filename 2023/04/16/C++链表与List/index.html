C++链表与list

发表于2023-04-16|更新于2023-04-16|C++
|阅读量:

一、介绍

STL中的List是数据结构中的双向链表，可以通过指针来进行数据的访问，它可以高效地再任意地方插入和删除，时间复杂度为常数级。主要用于插入和删除频繁，随机访问较少的场景。

二、应用

1.list的构造函数

构造函数 接口说明
list() 构造空的list
list(size_type n, const value_type& val = value_type()) 构造的list中包含n个值为val的元素
list(const list& x) 拷贝构造函数
list(InputIterator first, Inputlterator last) 用[first, last)区间中的元素构造list

例：

2.list的插入和删除

函数声明 接口说明
push_front(val) 在list首元素前插入值为val的元素
pop_front() 删除list中第一个元素
push_back(val) 在list尾部插入值为val的元素
pop_back() 删除list中最后一个元素
insert(int position, val) 在list中位置为position的位置插入一个值为val的元素
erase(int position) 删除在list中位置为2的元素

3.list迭代器的使用

函数声明 接口说明
begin + end 返回第一个元素的迭代器 + 返回最后一个元素的下一个位置的迭代器
rbegin + rend 返回第一个元素的reverse_iterator, 即end位置，返回最后一个元素下一个位置的reverse_iterator,即begin位置

其中begin 和 end为正向迭代器，rbegin 和 rend为反向迭代器

4.list获取头尾元素

函数声明 接口说明
front 返回list的第一个节点中值的引用
back 返回list的最后一个节点中值的引用

例：

5.获取list的大小

函数声明 接口说明
empty 判断list是否为空， 为空则返回true，否则返回false
size 返回list的长度（即链表的有效长度）

6.list的功能函数

函数声明 接口说明
swap(list l1, list l2) 用交换两个list容器的内容（即交换l1和l2的内容）
clear 用于清空整个容器，清空后容器的size为0
sort 将容器的数据排序（默认为升序）
resize 重设容器的大小
remove 移除指定元素
unique 去除list中重复的元素
reverse 逆转list容器，即翻转链表
